## 📋 זרימת העבודה המלאה:

### שלב 1: הורדת סרטון מיוטיוב
```
1. משתמש מכניס קישור YouTube
2. הקוד חולץ את ה-video ID
3. קורא ל-downloadYouTubeVideo()
4. מוריד את הסרטון דרך yt-dlp או Invidious
5. שומר את הקובץ ב-state
6. מציג את הנגן
```

### שלב 2: חיתוך מקטע
```
1. משתמש בוחר זמן התחלה וסיום
2. לוחץ על "חתוך ושמור"
3. הקוד קורא ל-cutVideo() או ל-Edge Function
4. חותך את המקטע
5. שומר ב-Storage
6. מוסיף לרשימת segments
```

### שלב 3: זיהוי שיר (אופציונלי)
```
1. משתמש לוחץ על "זהה שיר" על segment
2. הקוד מחלץ אודיו מהבלוב
3. שולח ל-Shazam API
4. מעדכן את שם ה-segment עם שם השיר
```

### שלב 4: מיזוג פלייליסט
```
1. משתמש מסדר את ה-segments (drag & drop)
2. לוחץ על "מזג הכל"
3. הקוד קורא ל-mergeSegments()
4. ממזג את כל ה-segments עם transitions
5. מוריד את הקובץ המאוחד
```
